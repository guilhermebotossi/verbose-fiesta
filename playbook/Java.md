
## Disassemble code from class

`javap -c <class>`

| param | desc                 |
| ----- | -------------------- |
| -c    | disassemble the code |

## Disassemble code from class

`javap -c <class>`

| param  | desc                        |
| ------ | --------------------------- |
| -v     | verbose output of the class |

## Extract classes from Jar

> `jar -x -f <jar_file>`

| param  | desc                          |
| ------ | ----------------------------- |
| -x     | extract classes from the jar  |
| -f     | indicates the file to be used |
